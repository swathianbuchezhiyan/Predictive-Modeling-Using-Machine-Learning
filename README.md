# Sales Prediction using Machine Learning

## Project Overview
This project predicts sales based on advertisement budget using Machine Learning (Linear Regression).

## Objective
- Analyze advertising data
- Build a prediction model
- Predict sales from TV, Radio, Newspaper ad budgets

## Dataset Features
- TV Ad Budget
- Radio Ad Budget
- Newspaper Ad Budget
- Sales

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Import libraries
2. Load dataset
3. Data cleaning
4. Data visualization
5. Define X and y
6. Train-test split
7. Model training (Linear Regression)
8. Prediction
9. Evaluation

## Model Used
Linear Regression

## Results
- Model predicts sales based on ad spending
- Evaluated using MAE and MSE
- Visualization: Actual vs Predicted graph

## Conclusion
This project shows how advertisement spending impacts sales using Machine Learning.
